ITServicesBusiness readme  Alex Chalmers 20260706
Working template of a website to advertize a business that provides IT services to small startups.
Uses MySql to store data and status of entered tickets entered by customers.
Employs JavaScript to presnt a GUI advertizing services and display status of entered tickets.
Users are customers and administrators.
Implements a simple workflow of registration then service requests by entering tickets. Tickets progress through states updated by admin : Open->In progress->Waiting->Resolved->Closed
Email used to notify originator of state changes. User can review status of tickets online.
Admins and Users see a dashboard of tickets appropriate for their id. 
Sample running is at www.fiberplayhouse.com/ITServicesBusiness
